# NPS Visitation Prediction Project – Stakeholder Emails

## Overview
This repository contains professional communication examples created for a scenario with the U.S. National Park Service (NPS).  
The NPS manages 63 National Parks and has seen a significant rise in visitation—312 million recreation visits in 2022.  
To address this, the NPS data team is building a machine learning model to predict future visitation at the most visited parks.  

The emails in this repository demonstrate how to tailor communication to different stakeholders:
- A **new data professional** joining the team (technical, workflow-focused).
- A **new public relations writer** (non-technical, mission-driven messaging).

---

## Contents
- `emails/email_to_data_professional.md`  
  Email tailored to a new data professional, focusing on project workflow, goals, and collaboration.  

- `emails/email_to_pr_writer.md`  
  Email tailored to a public relations writer, focusing on public-friendly messaging and the mission impact.  

- `templates/email_template_blank.md`  
  A blank template for professional stakeholder emails, reusable for future projects.  

---

## Skills Demonstrated
- **Audience analysis**: Identifying what each stakeholder needs and what is important to them.  
- **Professional communication**: Writing clear, concise, and purpose-driven emails.  
- **Stakeholder engagement**: Inviting follow-up questions to encourage collaboration and clarity.  
- **Documentation**: Organizing project communications in a structured, reusable format.  

---

## How to Use
1. Review the scenario in the README for context.  
2. Explore the completed emails to see how communication changes based on audience.  
3. Use the blank template to practice writing your own stakeholder emails for different projects.  

---

## Key Takeaway
Effective communication is just as important as technical expertise.  
By tailoring messages to different audiences, data professionals can ensure that technical projects are understood, supported, and celebrated across an organization.
